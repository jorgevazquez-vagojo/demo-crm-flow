Comparativa

Enginy vs HubSpot

Ambos CRMs estan soportados. La diferencia es en el metodo de integracion.

Caracteristica Enginy HubSpot
Metodo de integracion GenericCRMOutput + YAML HubSpotOutput dedicado
Configuracion config/crm/enginy.yaml .env (HUBSPOT_ENABLED + TOKEN)
Field mapping Declarativo en YAML Custom properties via API
Auth Bearer token Private app token
Dependencia extra Ninguna hubspot-api-client (pip)
Anadir nuevo CRM Solo YAML Requiere nuevo adapter Python
Cross-sell Via custom fields en YAML Via custom properties
Ideal para Pymes, CRMs custom Enterprise, equipos grandes

Tip: Puedes usar ambos simultaneamente. El pipeline exporta a todos los outputs activos.
